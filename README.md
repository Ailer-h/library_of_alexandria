# THIS PROJECT IS STILL A WORK IN PROGRESS
This project is a reading tracker, aiming to solve the problem of not having enough data on your reading periods.

## Used technologies.

- HTML and CSS;
- JavaScript;
- PHP;
- SQL;
- JQuery.

## Used softwares.

-   Xampp/Wamp - Hosting;
-   Git - Version Control;
-   VSCode - Code Editing;
-   phpMyAdmin - Database Management.
